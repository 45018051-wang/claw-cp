# USB OpenClaw (Portable Git Tool)

## Project Overview

USB OpenClaw is a lightweight portable Git tool designed specifically for the Windows environment. It can be run directly from a USB flash drive without requiring any installation on the system, delivering a true plug-and-play portable development experience.

## Key Features

- **Portable Operation**: No installation required; simply extract and use directly from a USB drive
- **Full Git Environment**: Includes all core components of Git for Windows
- **Graphical Control Panel**: Features clawpanel.exe for a graphical user interface
- **Multiple Shell Support**: Built-in support for common Unix shells such as bash and sh
- **Git GUI Tools**: Integrated visualization tools including git-gui and gitk
- **Credential Management**: Includes Git Credential Manager for secure credential storage
- **SSH Support**: Full SSH client and configuration integrated

## System Requirements

- Operating System: Windows 10 or later
- Architecture: 64-bit (x64)
- Storage Space: Approximately 500MB available space

## Quick Start

### Basic Operations

```bash
# Initialize a new Git repository
git init

# Clone an existing repository
git clone
```
